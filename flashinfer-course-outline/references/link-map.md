# FlashInfer 官方链接映射

这个文件提供 `flashinfer-course-outline` skill 常用的已核对官方链接。生成课程时，若要给“推荐阅读 / 推荐文档 / 推荐链接”，优先从这里取，不要临场猜 URL。

## 使用规则

- 这里只放官方站点或官方 GitHub 仓库中的稳定入口。
- 若用户要求“最新版本/指定版本”，仍需先核对版本，再决定是否引用这里的通用入口。
- 若某个主题在这里没有稳定链接，先退回到 [topic-map.md](./topic-map.md) 的主题索引，再视情况核对后补充。
